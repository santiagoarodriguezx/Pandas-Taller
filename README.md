## Descripción

Este proyecto contiene un taller práctico para aprender y reforzar el uso de **Series en Pandas**.  
A través de ejercicios progresivos se practican habilidades de creación, manipulación, análisis y transformación de datos unidimensionales.

## Contenido del taller

El notebook incluye ejercicios sobre:

1. Creación de Series a partir de listas
2. Creación de Series a partir de diccionarios
3. Creación de Series a partir de arrays de NumPy
4. Creación de Series con valores constantes
5. Uso de índices personalizados
6. Series con distintos tipos de datos
7. Atributos principales (`index`, `values`, `dtype`, `size`, `shape`, etc.)
8. Acceso a datos con `iloc` y `loc`
9. Operaciones aritméticas y manipulación
10. Aplicación de funciones (`apply`, `map`)
11. Ordenación y clasificación
12. Filtrado condicional
13. Manejo de valores nulos
14. Alineación, combinación y reindexación de Series
